# Telegram access bot (Docker)

## Что делает бот
1. Пользователь отправляет **свой** контакт кнопкой.
2. Бот возвращает текущий код.
3. Бот записывает вход в базу.

Админ‑панель (внутри бота):
- Изменение кода
- Список пользователей
- История входов

## Быстрый старт
1. Создайте `.env` рядом с `docker-compose.yml`:

```env
BOT_TOKEN=123456:ABCDEF
ADMIN_IDS=123456789
ACCESS_CODE=0000
DB_PATH=/data/bot.sqlite3
```

2. Запустите:

```bash
docker compose up -d --build
```

## Команды
- `/start` — отправка контакта
- `/admin` — админ‑панель
- `/setcode <новый_код>` — смена кода (админ)

## Примечания
- Код хранится в SQLite (таблица `settings`) и переживает перезапуски.
- История входов фиксируется по времени UTC.
